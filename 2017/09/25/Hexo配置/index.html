Hexo配置

发表于 2017-09-25 | 分类于 Hexo

1. 首页文章折叠

如果不想博文在首页全部显示, 并能出现阅读全文按钮效果, 需要在你想在首页显示的部分下添加

<!--more-->

2. 目录编号设置

修改主题配置文件（NexT.__config.yml）的number为false;

3. 主题更改

下载新的主题：

cd ~/git/hexo
git clone git@github.com:litten/hexo-theme-yilia.git themes/yilia

在./_config.yml，修改主题为yilia

theme: yilia

4. 部署到github

在配置文件__config.ym中加入：

deploy:
  type: github
  repo: https://github.com/lmintlcx/lmintlcx.github.io.git
  branch: master

5. 站点建立时间

例如 © 2014 - 2015

站点配置文件新增字段 since
since: 2014

6. 侧栏设置

post - 默认行为, 在文章页面(拥有目录列表)时显示
always - 在所有页面中都显示
hide - 在所有页面中都隐藏(可以手动展开)

sidebar: post

7. 头像设置

编辑主题配置文件, 新增字段 avatar, 头像的链接地址可以是:

网络地址
https://avatars2.githubusercontent.com/u/4962914?v=3&s=460

站点内的地址
/images/avatar.jpg #头像图片放置在主题的 source/images/

avatar: /images/avatar.png

8. 菜单设置

编辑主题配置文件的 menu
若站点运行在子目录中, 将链接前缀的 / 去掉

menu:
  home: /
  archives: /archives
  categories: /categories
  tags: /tags
  commonweal: /404.html
  about: /about

标签云页面
添加一个标签云页面, 并在菜单中显示页面链接

9. 新建tags页面

hexo new page “tags”

将页面的类型设置为 tags

title: tags
date: 2015-09-19 22:37:08
type: "tags"
---

关闭评论

title: tags
date: 2015-09-19 22:37:08
type: "tags"
comments: false
---
在菜单中添加链接。 编辑主题配置文件, 添加 tags 到 menu 中

menu:
  tags: /tags

10. 新建分类页面

添加一个分类页面, 并在菜单中显示页面链接

新建 categories 页面
hexo new page categories

将页面的类型设置为categories

title: categories
date: 2015-09-19 22:38:00
type: "categories"
---
关闭评论

title: categories
date: 2015-09-19 22:38:00
type: "categories"
comments: false
---

在菜单中添加链接. 编辑主题配置文件, 添加 categories 到 menu 中

menu:
  categories: /categories

11. 设置侧边栏默认打开

在主题配置文件中，设置：

display: always
